# Dynamic Wumpus Logic Agent

## Objective

This project implements a Knowledge-Based Agent that navigates a Wumpus World grid using propositional logic and resolution.

## Features

* Dynamic Grid
* Random Pits and Wumpus
* Percepts (Breeze, Stench)
* Knowledge Base (KB)
* CNF Conversion
* Resolution Refutation
* React UI Visualization

## Algorithm

1. Agent perceives environment
2. Updates KB
3. Converts rules into CNF
4. Uses resolution to infer safe cells

## Tech Stack

* React.js
* JavaScript

## Deployment

Deployed using Vercel.

## Challenges

* CNF conversion complexity
* Resolution performance
* Logical consistency

## Output

* Safe path navigation
* Real-time inference tracking
